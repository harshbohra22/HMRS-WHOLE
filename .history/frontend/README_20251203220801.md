## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## API Integration

The frontend connects to the backend API at `https://hmrs.onrender.com/api`. All API calls are centralized in the `services/api.ts` file.

## Features in Detail

### Job Search & Filtering
- Search by job title or company name
- Filter by city
- Show only active jobs
- Sort by deadline
- Real-time filtering

### Registration
- Separate forms for employers and job seekers
- Form validation with helpful error messages
- Password confirmation
- Success/error notifications

### Dark Mode
- Toggle dark/light mode
- Persists preference in localStorage
- Respects system preference on first visit

## Environment Variables

Currently, the API URL is hardcoded. To use environment variables:

1. Create a `.env` file:
```
VITE_API_URL=https://hmrs.onrender.com/api
```

2. Update `services/api.ts`:
```typescript
const API_BASE_URL = import.meta.env.VITE_API_URL || 'https://hmrs.onrender.com/api';
```
